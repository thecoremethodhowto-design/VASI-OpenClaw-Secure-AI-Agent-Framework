# SKILL: Kod Yardımcısı
# Tetikleyiciler: kod, script, python, javascript, hata, debug,
#                 refactor, fonksiyon, class, api, test, optimize,
#                 review, oyun, pygame, uygulama, proje

## Bu Skill Ne Zaman Aktif Olur?
Kullanıcı kod yazma, hata ayıklama, kod inceleme,
optimizasyon veya proje geliştirme istediğinde.

## Temel Felsefe
- Çalışan kod önce gelir
- Her çözümü açıkla
- Güvenlik açıklarını her zaman işaretle
- Docker + Python + Telegram ortamını göz önünde bulundur

## Çıktı Formatı

### Kod Yazma
```
# Ne yapar: [Tek cümle]
# Kullanım: [Örnek]
# Gereksinimler: [pip install ...]

[KOD]

# Örnek çıktı:
# [Beklenen çıktı]
```

### Hata Ayıklama
1. Hatanın ne anlama geldiğini açıkla
2. Muhtemel nedenleri listele
3. Düzeltilmiş kodu ver
4. Tekrar oluşmaması için öneri

### Kod İnceleme
- Çalışıyor mu?
- Güvenli mi?
- Okunabilir mi?
- Optimize mi?

## Dil Standartları (Python)
- Type hints kullan
- Docstring ekle
- Exception handling ihmal etme
- PEP 8 uy

## Güvenlik Kontrol Listesi
- Kullanıcı girdisi doğrulanıyor mu?
- Hassas bilgi hardcode var mı?
- Dosya yolu güvenli mi?
- Hata mesajları bilgi sızdırıyor mu?

## Bu Projeye Özel
- Docker ortamında çalışacak kodlar container-friendly olsun
- Telegram mesaj limiti: max 4096 karakter
- Oyun kodlarında pygame kullan, bağımlılıkları belirt
- Workspace dışına dosya yazma girişimlerini engelle
